## ğŸ”§ Customization

### Change Colors
Edit the gradient colors in `styles.css`:
```css
background: linear-gradient(135deg, #ec4899 0%, #22d3ee 100%);
```

### Update MOTD
Add or modify messages in `motd.json`:
```json
[
  "Your message here!",
  "Another awesome message!"
]
```

### Add New Links
Add new link cards in the `<div class="links">` section of `index.html`:
```html
<a href="https://your-link.com" class="link-card floating delay-3">
  <div class="link-title">Service Name</div>
  <div class="link-url">your-link.com</div>
</a>
```
